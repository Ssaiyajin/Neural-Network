# Neural-Network

A simple **from-scratch neural network demo** using the `nnfs` helper dataset.  
Primary implementation can be found in: [`Neural-Network/neural_network.py`](Neural-Network/neural_network.py)

---

## Overview

This project implements a minimal **feed-forward neural network** with categorical cross-entropy loss:

- **Layers**: [`Layer_Dense`](Neural-Network/neural_network.py)  
- **Activations**: [`Activation_ReLU`](Neural-Network/neural_network.py), [`Softmax`](Neural-Network/neural_network.py)  
- **Loss**: [`Loss_CategoricalCrossentropy`](Neural-Network/neural_network.py)  
- **Data**: `spiral_data` from the `nnfs` package  

The project is designed for educational purposes and interactive exploration of neural network fundamentals.

---

## Requirements

- Python 3.8 or higher  
- `numpy`  
- `nnfs`  
- (Optional) Jupyter Notebook or JupyterLab for interactive experimentation  

---

## Installation

Install the required packages using pip:

```bash
pip install numpy nnfs

```
